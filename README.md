
코딩 테스트 / 알고리즘 문제 풀이 코드 편집기

## 기능
*   **사용 가능 언어:** C++, Python, Java
*   Syntax Highlighting만 제공 (자동 완성 및 에러 밑줄 X)
*   여러 개의 테스트 케이스(Input/Expected Output) 관리
*   파일 드래그 드롭 지원



<br>

### 언어 / 프레임워크
- **Core:** Electron
- **Frontend:** React, Vite, Tailwind CSS
- **Store:** electron-store (설정 및 코드 영속성)
- **Editor:** Monaco Editor
- **AI Assistance:** Gemini-CLI


<br>

---


### 1. 필수 요구사항
*   Node.js (v16 이상 권장)
*   각 언어별 컴파일러/인터프리터 (g++, python3, javac/java)

### 2. 프로젝트 설치
```bash
npm install
```

### 3. 개발 모드로 실행
```bash
npm run dev
```

### 4. 빌드 (실행 파일 생성)
```bash
npm run build
```
빌드된 파일은 `dist` 또는 `release` 폴더에 생성됨.

---

### ⚙️ 초기 설정 (컴파일러 경로)

앱을 처음 실행한 후, 우측 상단의 **⚙️ Settings** 버튼을 눌러 각 언어의 컴파일러 경로 설정
